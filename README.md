# UVA Accepted Solutions

This repository contains my **accepted solutions** to problems from the **University of Valladolid Online Judge (UVa Online Judge)**.

## About

The purpose of this repository is to:

- Store accepted UVa submissions for future reference.
- Review algorithms and data structures used in competitive programming.
- Track problem-solving progress.

Each solution has been accepted by the UVa Online Judge at the time of submission.

## Repository Structure

Solutions are organized by problem number and/or title, depending on the folder structure.

Example:

```
100 - The 3n + 1 Problem.cpp
101 - The Blocks Problem.cpp
102 - Ecological Bin Packing.cpp
...
```
